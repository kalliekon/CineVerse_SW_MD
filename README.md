# CineVerse_SW_MD
CineVerse is an XML-based movie database template created with Altova XMLSpy. It models movies, directors, actors and awards with structured relationships. Data is validated using DTD and transformed to HTML with XSLT. All images belong to their respective owners and are used only for portfolio demonstration.
